# Photos, logo & poster go here

Drop files with these exact names — the site loads them automatically. Until a file
exists, a clean placeholder shows and nothing breaks.

## Logo (header, auto-swaps by style)
- `logo-light.svg` — your **white** logo (shows on Noir / dark background)
- `logo-dark.svg` — a **dark** version (shows on Mono / Signal / light background)
- PNG also fine (`logo-light.png` / `logo-dark.png`) — see the note in ../README.md

## Photography
- `work-1.jpg` — big wide featured tile (strong landscape image)
- `work-2.jpg … work-5.jpg` — the rest of the "Recent frames" grid + Index hover-previews

## Optional
- `reel-cover.jpg` — poster behind the showreel play button (1920×1080)
- `og.jpg` — social-share preview (1200×630)

Tip: you can save images from your old Wix site (right-click → Save image),
rename them to match the list, and drop them in here.
