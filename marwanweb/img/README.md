# Photos go here

Drop your images in this folder using these exact names — they load automatically:

- `marwan.jpg` — hero portrait (~1000×1250, portrait / 4:5)
- `artist.jpg` — "Marwan the Artist" card (~1200×750)
- `broker.jpg` — "Marwan the Broker" card (~1200×750)
- `director.jpg` — "Creative Director" card (~1200×750)
- `photo-1.jpg` … `photo-6.jpg` — photography gallery (any size; portrait looks best)
- `reel-cover.jpg` — optional poster behind the showreel play button (1920×1080)
- `og.jpg` — social-share preview (1200×630)

Tip: you can save the images from your old Wix site (right-click → Save image),
rename them to match the list above, and drop them in here.
Until a file exists, the site shows a clean placeholder — nothing breaks.
