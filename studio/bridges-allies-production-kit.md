# 🎬 Bridges & Allies — Production Kit
**60-second multilingual partner-invitation video · Studio (Smart AI Team)**

Goal: invite international real-estate agents/partners to join your Dubai
network. Follow each scene block, tick the asset checklist, assemble in CapCut.

---

## ⚙️ Global settings
- **Length:** 60s · 8 scenes
- **Format:** shoot/export **9:16 vertical** (Reels/TikTok/Shorts/LinkedIn) —
  optional **16:9** version for website/YouTube
- **Look:** premium, warm, aspirational; gold-hour Dubai; clean corporate
- **Music:** uplifting cinematic corporate, subtle build to the closing logo
- **Captions:** burned-in subtitles **in each scene's language + English** (key
  for a multilingual recruiting video — viewers watch muted)
- **Brand:** Bridges & Allies logo on closing; logo bug bottom-corner throughout
- **Recommended toolchain:** HeyGen (talking scenes) + stock/AI b-roll + CapCut
  (assembly, captions, music)

## 🚀 Build it this week (5 steps)
1. **HeyGen** account → create the speaking scenes: pick avatars, paste each
   language's VO line, generate. (Linguist can polish translations first.)
2. **Gather b-roll** for scenes 1, 6, 8 from Pexels (free) — search "Dubai
   aerial", "desert safari", "Dubai sunset".
3. **ElevenLabs** for any voiceover-only lines (S6) if not using HeyGen voices.
4. **CapCut**: drop clips on the timeline in scene order, add music, auto-
   captions (translate per scene), logo bug, and the closing end card.
5. **Export 9:16**, review, then hand to **Recruiter/Scribe** to publish + DM
   target agents in each region.

## ⚠️ Before you publish
- **Verify the "5% tax" line (Scene 4).** Dubai has 0% personal income tax; a
  flat "5%" could mislead and hurt credibility. Either cut the tax claim or
  state the exact, correct figure for what you mean (e.g. 4% DLD transfer fee,
  5% VAT on commercial). Get this right — it's a recruiting trust signal.
- Confirm you have rights/consent for any real people filmed, and licenses for
  stock/music.
