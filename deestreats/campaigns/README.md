# Campaigns

One folder per campaign. Each campaign keeps its brief, shot lists, and copy together.

## Active
- **`world-cup/`** — World Cup–themed set across all six menu items. Three-shot structure per item (Stadium → Trophy/Walkout → Showcase), cinematic and consistent for cohesion. Meta, Send Message objective, Dubai-targeted.

## Rules (apply to every campaign)
- **CTA:** every cut ends on a clear *Send Message / DM to order* prompt.
- **Mute-first:** on-screen supers on every shot — most viewers watch silent.
- **Brand early:** brand + offer in the first 1–2 seconds, not the end.
- **Legal:** no real footballer likenesses or named athletes. Generic kit descriptions only (see world-cup brief).
- **Fix, don't reshoot:** recut existing footage with supers + CTA before shooting anything new.
