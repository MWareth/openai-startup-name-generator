# Dee's Treats — Working Folder

Home for all Dee's Treats marketing/creative work. The brand and voice rules live in the root [`CLAUDE.md`](../CLAUDE.md) — read that first.

## Folders
- **`campaigns/`** — one folder per campaign. `world-cup/` holds the six-item, three-shot (Stadium → Trophy/Walkout → Showcase) World Cup set.
- **`copy/`** — hooks, CTAs, captions, DM/WhatsApp scripts. Voice: short, punchy, direct.
- **`brand/`** — colors (brand-brown, cream), fonts (NF One Little Font Bold for ad supers, Lemonade for video supers), and any brand reference.
- **`assets/`** — exports, shot lists, reference frames. Keep heavy raw video out of git; store finals/references only.

## How to get back here later
Just say **"open the Dee's Treats folder"** or **"work in `deestreats/`"** and Claude will pick up here. From a terminal: `cd deestreats`. Each subfolder keeps a `.gitkeep` so the structure survives even when empty.
