# Photos for the website

Drop your photos into this `img/` folder using these **exact file names** and the
website will use them automatically. Until a photo exists, a colored tile with an
emoji shows in its place — so the site always looks complete.

## Menu photos (square or 4:3 works best)
- `pavlova.jpg`
- `sansebastian.jpg`
- `lazycake.jpg`
- `carrotballs.jpg`
- `carrotcake.jpg`
- `cremebrulee.jpg`
- `bundle.jpg`

## Story strip (4 small square photos)
- `story1.jpg`  `story2.jpg`  `story3.jpg`  `story4.jpg`

## Gallery (up to 8 square photos)
- `gallery1.jpg` … `gallery8.jpg`

Tip: `.jpg` names are expected. Keep files reasonably small (under ~500 KB each)
so the site loads fast on phones.
