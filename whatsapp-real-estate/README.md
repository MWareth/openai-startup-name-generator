## Owner's-update sequence (same for every community)
Each report follows the format proven on The Acres:
1. **Hero** — community render/photo + headline.
2. **At a glance** — 4 key tiles (recent sales count, avg price, avg AED/sq ft, YoY).
3. **Recent sales** — real DLD transfers table by cluster/sub-location.
4. **Price trend** — interactive Property Monitor index chart.
5. **Construction / community update** — status, handover or handed-over context.
6. **Sell vs hold** — honest, side-by-side owner guidance.
7. **Free valuation CTA** — opens Marwan's WhatsApp (+971 56 164 5645).

Built from each community's Property Monitor / DLD report — real transfers, not estimates.
