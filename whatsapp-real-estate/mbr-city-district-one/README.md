# MBR City / District One — WhatsApp Outreach (standalone)

A separate outreach sender for the District One (Mohammed Bin Rashid Al Maktoum City)
buyer-match campaign. Fully isolated from the Acres tool — its own message presets,
its own contact box, and its own sent-tracking (localStorage keys `mbr-wa-*`), so the
two campaigns never mix.

- `whatsapp-outreach-mbr.html` — the tool. Presets: buyer match (2-3 bed), short, scarcity, free valuation.
- Contacts: paste `Name, +phone` per line (or a spreadsheet paste). Nothing preloaded.
